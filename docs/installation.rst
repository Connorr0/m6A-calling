============
Installation
============


Stable release
--------------

To install m6A-calling, run this command in your terminal:

.. code-block:: console

    $ pip install m6a_calling

This is the preferred method to install m6A-calling, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for m6A-calling can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/mrvollger/m6a_calling

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/mrvollger/m6a_calling/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/mrvollger/m6a_calling
.. _tarball: https://github.com/mrvollger/m6a_calling/tarball/master
